# FAQ (Tanya Jawab Tersering) & Solusi Kendala

Jangan panik bila memunculkan pesan merah di layar, sebagian besar bisa dipecahkan sendiri.

## Q: Aplikasi saya bilang "Perangkat ini sudah terdaftar untuk akun lain!" (Duplicate Device / Device Binding Error)
**Jawaban/Solusi:**
Aplikasi kita sangat ketat. 1 Akun Pegawai = 1 HP Murni. Jika Anda dulu pernah *login* pakai HP Samsung Anda, lalu HP itu rusak dan Anda baru saja menginstal aplikasi ini ke HP Xiaomi baru Anda, maka Server mendeteksi kecurangan.
**Tindakan:** Hubungi Admin/HRD untuk mereset *Device ID* Anda dari sistem Web Server. Setelah itu Anda bisa masuk seperti biasa di HP baru Anda.

## Q: Pemindai Sidik Jari (Biometrik) di aplikasi gagal memindai terus menerus padahal di lock screen HP saya lancar?
**Jawaban/Solusi:**
Sensor sidik jari bisa error saat jari basah/kotor. 
Jika berlarut-larut terus, tekan tombol **Beralih ke PIN**. Masukkan 6 digit keamanan PIN sebagai alternatif yang sah dan cepat.

## Q: Ada pesan merah "GPS Wajib Aktif untuk Melakukan Absensi" atau titik saya salah.
**Jawaban/Solusi:**
1. Anda mematikan fitur 'Lokasi' di panel atas (*notification bar*) HP Anda. Nyalakan itu!
2. Jika Titik Lokasi Peta di aplikasi kosong:
   - Pastikan sinyal luar ruangan cukup bagus agar HP menangkap sinyal satelit.
   - Buka Pengaturan HP > Aplikasi > Presensi Karyawan > *Permissions* (Izin) > Pastikan "Location" tertulis **Allow all the time** atau **Allow only while using the app**.

## Q: Tadi saya absen Offline dengan tulisan "Disimpan ke memori HP". Kok sekarang pas ada WiFi tidak terkirim-kirim?
**Jawaban/Solusi:**
Tarik-tahan layar Beranda Anda ke bawah *(Pull to Refresh)* sampai bola pemuatan *(loading)* bulat berputar di atas. Sistem kami akan segera mengecek tumpukan memori HP dan menembakkan paksa absen *Offline* Anda ke server.
